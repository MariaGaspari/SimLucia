This GitHub contains the core code and *some* microdata to make the user familiar with the IPF-based spatial microsimulation method used in the paper:
Unveiling Vulnerabilities: Integrating Socioeconomic and Physical Dimensions in Residential Exposure Modelling, published in the International Journal of Disaster Risk Reduction in 2025.

It contains the SimLucia model (SimLucia.R), a spatial microsimulation model for residential building exposure development incorporating key structural and economic dimensions of vulnerability, and its inputs (i.e., a sample of anonymised household-level data (MicrodataSample.csv) and the 2010 census-based datasets (Census_HHLD.csv)).

Since the actual microdata sample from the 2016 Living Conditions and Household Budgets Survey used in this paper is confidential, we cannot make them available. To access the data, please contact the Central Statistical Office of Saint Lucia.
